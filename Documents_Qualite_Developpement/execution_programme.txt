Ce projet a été fait avec nuxt.js, donc il ne peut y avoir d'exécutable.

Lancer le projet simplement :

- Se rendre dans le répertoire cloné.
- npm install
- npm run dev

Pour lancer les tests unitaires :

- Toujours dans le répertoire cloné.
- Le programme ne doit pas être lancé avec `npm run dev` en même temps
- npm run test
